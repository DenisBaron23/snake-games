# snake-games
[![Maintainability](https://qlty.sh/gh/DenisBaron23/projects/snake-games/maintainability.svg)](https://qlty.sh/gh/DenisBaron23/projects/snake-games)

Snake Game  — это современная реализация классической игры в змейку с расширенным функционалом. Игра поддерживает четыре уровня сложности, систему достижений, звуковые эффекты и полное сохранение статистики. Разработана на чистом JavaScript с использованием Canvas API для максимальной производительности и совместимости со всеми современными браузерами.

Стек
HTML5/CSS3/Vanilla JavaScript (ES6+)/Canvas 2D/Web Audio API/LocalStorage/Git/GitHub/Code Climate
